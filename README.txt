Yeast Divison Analysis Script
=============================

This program calculates moving averages for a test group of lifespan division counts and compares them to WT group using a t-test.


Usage
-----

* Add your XLS data files to the `input` folder.
* Edit process.py to set `window` and `windowSpan` for calculations.
* Run `python process.py`

Or use the GUI to pick input directory and window parameters

* Run `python process_gui.py`
